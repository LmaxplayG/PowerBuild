# PowerBuild

Build MSVC projects using only PowerShell 7.2+.

## Requirements

### Required

- PowerShell 7.2+
- MSVC 143 (VS 2022)
- Any Windows 10/11 SDK (10.0.19041.0+, PowerBuild will use the latest one installed by default)

### Recommended

- PowerShell 7.3+
- Any Windows 11 SDK
- 64-bit Host OS
